Projects

These are my personal projects, with links to their latest releases. Each of these programs is free software.

The word "free" in "free software" refers to freedom: my programs respect your freedom to run, copy, study, change, and redistribute them for any purpose provided that you extend those same freedoms to others. For the exact legal terms and conditions, refer to the appropriate license information that is distributed with the respective program.

Nonny

Play and create nonogram puzzles! Nonograms are logic puzzles in which numeric clues are used to fill in squares within a grid in order to reveal a picture.

Latest version:   Nonny 0.1, released 2017-06-14.

ccalc

Evaluate mathematical expressions from the command line. Accepts and evaluates expressions using C syntax.

Latest version:   ccalc 1.02, released 2017-05-25.

Option++

C++ facility for reading command-line program options. Provides an easy way to read, validate, and process command-line options.

Latest version:   Option++ 1.01, released 2017-05-26.

Nonogram Studio

Coming soon: Play and create nonogram puzzles! Nonogram Studio will retain all of the major features of Nonny while offering a completely redesigned user interface and many other improvements.

Latest version:   Unreleased.
